This folder contains MATLAB Code, Figure and Table Parameters to generate all the graphs in research paper submitted in Congent Engineering Journal on the topic of Robust User Identification in Massive MIMO Systems Using Compressed Sensing Techniques.

Use Main Code.m to generate all the plots/images given in the paper and use the parameter given in Tables.docx file to generate remaining plots for all the scenarios.

Main Code.m will generate Figure 3-Figure 6 for remaining figures Main Code 1.m to Main Code 7.m can be used or else follow the process given in second paragraph.
